# AWS Advocacia — Landing Page

Site institucional do escritório **Adriana e Walter Soares Advocacia**, desenvolvido com HTML, CSS e JavaScript puro, hospedado via GitHub Pages.

## Acesse o site

[awsadvocacia.adv.br](https://awsadvocacia.adv.br)

---

## Sobre o projeto

Landing page mobile-first desenvolvida para apresentar o escritório, suas áreas de atuação e facilitar o contato de potenciais clientes. O projeto foi construído sem frameworks ou dependências externas — apenas HTML semântico, CSS com variáveis customizadas e JavaScript vanilla.

## Seções

- **Nav** — Logo + menu hamburguer responsivo
- **Hero** — Apresentação do escritório com CTA
- **Áreas de atuação** — 6 cards com as especialidades
- **Nossos advogados** — Perfil do Dr. Walter e Dra. Adriana
- **Contato** — Formulário via Formspree
- **CTA** — Link direto para WhatsApp
- **Footer** — Copyright

---

## Tecnologias

- HTML5 semântico
- CSS3 — variáveis customizadas, Flexbox, Grid, media queries
- JavaScript vanilla — menu hamburguer, IntersectionObserver
- [Formspree](https://formspree.io) — envio de formulário sem back-end
- GitHub Pages — hospedagem gratuita com HTTPS

## Como rodar localmente

```bash
git clone https://github.com/seuusuario/aws-advocacia.git
cd aws-advocacia
# Abre o index.html no navegador
# ou usa o Live Server do VS Code
```

Não há dependências para instalar — abre o `index.html` direto no navegador.
